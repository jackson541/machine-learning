# Regressão Linear
A regressão linear é uma técnica que busca encontrar uma função linear (do tip A + Bx) para relacionar variáveis independentes com uma dependente.

Ex: Encontrar uma função que preveja o valor de um imóvel com base no seu tamanho e na sua idade.
